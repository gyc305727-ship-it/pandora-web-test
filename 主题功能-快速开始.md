# 主题切换功能 - 快速开始

## 🚀 30秒快速体验

### 1. 启动项目
```bash
cd web
npm run dev
```

### 2. 访问系统
打开浏览器访问：`http://127.0.0.1:3000`

### 3. 体验主题切换
1. 登录系统
2. 在顶部导航栏右上角找到月亮/太阳图标
3. 点击图标切换主题
4. 观察整个系统的颜色变化

## 🔍 快速验证

打开浏览器开发者工具（F12）：

1. **查看 localStorage**  
   Application → Local Storage → 查看 `pandora-theme` 键

2. **查看 HTML 类名**  
   Elements → `<html>` 标签 → 暗色模式时有 `dark` 类

3. **查看 CSS 变量**  
   Elements → 选中任意元素 → Computed → 搜索 `--bg-color`

## ❓ 常见问题

**Q: 切换后部分区域颜色没变？**  
A: 刷新页面，确保 `theme.css` 已加载。

**Q: 刷新后主题恢复默认？**  
A: 检查浏览器是否禁用了 localStorage。

**Q: 如何修改主题颜色？**  
A: 编辑 `web/src/assets/styles/theme.css` 中的 CSS 变量。
